##  Featured Projects

###  Fraud Detection Under Class Imbalance
 https://colab.research.google.com/drive/17mdk9ZJ-__BzIiN_Ge-fKrbDOPGwZeT0?usp=sharing

Worked on a highly imbalanced fraud dataset (~0.4% fraud rate).

Compared:
- Logistic Regression
- Class weights
- Undersampling
- SMOTE

**Key Insight:**
ROC-AUC can be misleading in imbalanced datasets
PR-AUC gives more realistic performance insight
